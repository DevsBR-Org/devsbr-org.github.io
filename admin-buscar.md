# Buscar Jogadores e Personagens

Ferramenta de busca unificada para encontrar contas de usuários e personagens no banco de dados do jogo.

---

## Como Acessar

1. Faça login no GameCP com sua conta admin
2. Clique em **Admin** no menu lateral
3. Clique na aba **Buscar**

---

## Buscar por Usuário

1. Clique no botão **"Usuários"** na seleção de tipo de busca
2. No campo de busca, digite o **login** da conta
3. Pressione Enter ou clique em **"Buscar"**
4. Os resultados aparecem abaixo

### Listas rápidas

Use os botões de atalho para listar:

| Botão | O que faz |
| --- | --- |
| **Lista de Logins** | Exibe todos os logins do servidor (paginado) |
| **Lista de Personagens** | Exibe todos os personagens do servidor |
| **Listar Contas Banidas** | Mostra apenas contas banidas |
| **Chars Excluídos** | Mostra personagens que foram excluídos |

---

## Buscar por Personagem

1. Clique no botão **"Personagens"** na seleção de tipo de busca
2. No campo de busca, digite o **nome do personagem** ou **serial**
3. Pressione Enter ou clique em **"Buscar"**
4. Clique em um personagem nos resultados para ver os detalhes
