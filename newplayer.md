# New Player (Novo Jogador)

O sistema New Player permite criar campanhas que dão recompensas para novos jogadores que cumpram determinados requisitos.

---

## Como Acessar

1. Faça login no GameCP com sua conta admin
2. Clique em **Admin** no menu lateral
3. Clique na aba **New Player**

---

## Criar uma Nova Campanha

1. Clique no botão **"Nova"** na lista de campanhas (coluna esquerda)
2. Preencha os campos:

| Campo | O que preencher |
| --- | --- |
| **Nome da Campanha** | Nome identificador (ex: "Boas-vindas Março 2026") |
| **Campanha Ativa** | Marque o toggle para ativar |
| **Descrição** | Descrição da campanha para os jogadores |

---

## Configurar Restrições

As restrições definem quais jogadores podem participar. Para cada restrição, **marque o checkbox** para ativá-la e defina o valor:

| Restrição | O que faz |
| --- | --- |
| **Data de Criação do Personagem** | Personagem precisa ter sido criado após esta data |
| **Level Mínimo** | Personagem precisa ter pelo menos este level (1-66) |
| **PvP Point Mínimo** | Personagem precisa ter no mínimo este PvP Point |
| **Tempo de Jogo Mínimo** | Personagem precisa ter jogado pelo menos X minutos |
| **Dalant Mínimo** | Personagem precisa ter pelo menos esta quantidade de Dalant |
| **Processing Point Mínimo** | Ponto mínimo de processamento |
| **Hunting Point Mínimo** | Ponto mínimo de caça |
| **Gold Point Mínimo** | Ponto mínimo de gold |

Clique em **"Salvar Campanha"**

---

## Configurar Recompensas

Após salvar a campanha, a seção de **recompensas** aparece abaixo. Configure os itens que serão entregues por raça usando as abas disponíveis.

---

## Editar uma Campanha

1. Clique na campanha desejada na lista da esquerda
2. Altere os campos necessários
3. Clique em **"Salvar Campanha"**

---

## Excluir uma Campanha

1. Selecione a campanha na lista
2. Clique em **"Excluir"**
3. Confirme a exclusão
