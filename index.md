## Informações Técnicas

### Versão Atual

- **Versão:** 1.6.0
- **Ultima atualização:** Março 2026

### Requisitos

- WordPress 5.0+
- PHP 8.2+
- WooCommerce 5.0+
- SQL Server (RF Online)
- Extensão PDO SQLSRV ou DBLIB

### Stack Tecnológica

- PHP 8.2+, Bootstrap 5, jQuery
- Integração WooCommerce nativa
- API REST para módulos internos

### Sistemas de Proteção Suportados

- Odin
- DevCorp
- HGK
- SirinGuard
