# Dashboard Admin

O Dashboard é a tela inicial do painel admin, com estatísticas do servidor e log de todas as ações administrativas.

---

## Como Acessar

1. Faça login no GameCP com sua conta admin
2. Clique em **Admin** no menu lateral
3. O Dashboard é a primeira aba exibida

---

## Estatísticas

Na parte superior, quatro cards mostram:

| Card | O que exibe |
| --- | --- |
| **Total de Contas** | Quantidade total de contas registradas |
| **Total de Personagens** | Quantidade total de personagens criados |
| **Jogadores Online** | Quantidade de jogadores conectados no momento |
| **Administradores** | Quantidade de admins cadastrados |

---

## Últimas Ações (Log)

A tabela de log mostra todas as ações executadas por administradores.

### Filtrar o log

1. **Admin** — Selecione um admin específico ou "Todos"
2. **Ação** — Filtre por tipo de ação ou "Todas"
3. **Data Início / Data Fim** — Defina o intervalo de datas
4. Clique em **"Filtrar"** ou os filtros de select aplicam automaticamente
5. Clique em **"Limpar"** para remover todos os filtros

### Colunas do log

| Coluna | O que exibe |
| --- | --- |
| **Data/Hora** | Quando a ação foi executada |
| **Admin** | Quem executou a ação |
| **Ação** | Tipo da operação (badge colorido) |
| **Detalhes** | Informações específicas da ação |

> O dashboard usa cache de **5 minutos**. Clique em **"Atualizar"** para ver as ações mais recentes.
