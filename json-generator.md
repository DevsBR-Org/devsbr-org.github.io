# JSON Generator (Gerar JSON dos Itens)

Gera arquivos JSON a partir do banco de dados de itens do jogo, usados pelo sistema de autocomplete e busca de itens.

---

## Como Acessar

1. Faça login no GameCP com sua conta admin
2. Clique em **Admin** no menu lateral
3. Clique na aba **JSON Generator**

---

## Gerar Todos os JSONs

1. Clique em **"Gerar Todos os JSON"**
2. Acompanhe o progresso no console de log
3. A página recarrega após a geração

---

## Gerar por Tipo

Se preferir, clique no botão do tipo específico de item:

- upper, lower, gauntlet, shoe, helmet, shield, weapon, cloak, ring, amulet
- bullet, maketool, bag, potion, face, force, battery, ore, resource
- unitkey, booty, map, town, battledungeon, animus, guardtower, trap
- siegekit, ticket, event, recovery, box, firecracker, unmannedminer, radar, npclink

---

## Gerenciar Arquivos Gerados

Na tabela inferior:

- Veja o **nome**, **tamanho** e **data de modificação** de cada arquivo
- **Excluir individual** — clique no botão de excluir de cada arquivo
- **Excluir Todos** — remove todos os JSONs gerados
